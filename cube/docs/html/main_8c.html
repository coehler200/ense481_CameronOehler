Detailed Description

Program to read from IMU and send over BLE to remote renderer.

Author
Cameron Oehler

Cube Visualizer is a program developed for the embedded system STM32F103RB. The program is an RTOS which reads data from a 9DOF sensor and sends this data over BLE to a visual renderer on another system. There is also a USART command line interface for interacting with the system.
